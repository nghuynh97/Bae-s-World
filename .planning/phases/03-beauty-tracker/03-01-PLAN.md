3. Create `scripts/seed-beauty.ts`:

```typescript
import { db } from "@/lib/db";
import { beautyCategories, routines } from "@/lib/db/schema";

async function seed() {
  console.log("Seeding beauty categories...");
  const defaultCategories = [
    { name: "Skincare", slug: "skincare", displayOrder: 0, isDefault: 1 },
    { name: "Makeup", slug: "makeup", displayOrder: 1, isDefault: 1 },
    { name: "Haircare", slug: "haircare", displayOrder: 2, isDefault: 1 },
    { name: "Body Care", slug: "body-care", displayOrder: 3, isDefault: 1 },
  ];
  for (const cat of defaultCategories) {
    await db.insert(beautyCategories).values(cat).onConflictDoNothing();
  }
  console.log("Beauty categories seeded.");

  console.log("Seeding default routines...");
  const defaultRoutines = [
    { name: "Morning", slug: "morning", displayOrder: 0 },
    { name: "Evening", slug: "evening", displayOrder: 1 },
  ];
  for (const routine of defaultRoutines) {
    await db.insert(routines).values(routine).onConflictDoNothing();
  }
  console.log("Default routines seeded.");

  process.exit(0);
}

seed().catch((e) => { console.error(e); process.exit(1); });
```

1. Create `src/actions/beauty-products.ts` with these Server Actions following the portfolio.ts pattern:

**getBeautyProducts(categorySlug?: string, favoritesOnly?: boolean)**
- NO auth gate (page.tsx will gate access; but actually BEAU-07 requires auth, so ADD auth gate)
- Auth gate: createClient() -> getUser() -> throw "Unauthorized"
- Query: select from beautyProducts innerJoin beautyCategories innerJoin images
- If categorySlug provided and not "all" and not "favorites": filter by beautyCategories.slug
- If favoritesOnly is true OR categorySlug === "favorites": filter isFavorite === 1
- Order by desc(beautyProducts.createdAt)
- After fetching products, batch-fetch imageVariants using inArray(imageVariants.imageId, imageIds)
- Use getSignedImageUrls from "@/lib/supabase/storage" to sign variant paths (private bucket)
- Map signed URLs back to variants
- Return products with their signed variant URLs

**getBeautyProductById(productId: string)**
- Auth gate
- zod UUID validation
- Fetch single product with category join + image join
- Fetch variants and sign URLs
- Return product with variants or null

**createBeautyProduct(data: { name, brand?, categoryId, rating, notes?, imageId })**
- Auth gate
- zod schema: name string min(1) max(100), brand string max(100) optional nullable, categoryId uuid, rating number int min(0) max(5), notes string max(1000) optional nullable, imageId uuid
- Insert into beautyProducts, return created record

**updateBeautyProduct(productId: string, data: { name?, brand?, categoryId?, rating?, notes? })**
- Auth gate, UUID validation
- zod schema with all optional fields
- Set updatedAt: new Date()
- Return updated record

**deleteBeautyProduct(productId: string)**
- Auth gate, UUID validation
- Find product to get imageId
- Get variant storage paths
- Delete from supabase.storage.from("private-images").remove(paths)
- Delete image record (cascades to variants)
- Delete product record
- Return { success: true }

**toggleFavorite(productId: string)**
- Auth gate, UUID validation
- Fetch product, toggle isFavorite (0 -> 1, 1 -> 0)
- Set updatedAt: new Date()
- Return updated product

**searchBeautyProducts(query: string)**
- Auth gate
- Search beautyProducts where name ilike `%query%`
- Return array of { id, name, brand, imageId } (lightweight for routine step search)
- Limit 10 results

2. Create `src/actions/beauty-categories.ts` following categories.ts pattern:

**getBeautyCategories()**
- Auth gate
- Select all from beautyCategories ordered by displayOrder asc
- Return array

**createBeautyCategory(data: { name: string })**
- Auth gate
- zod: name min(1) max(50)
- Generate slug (same generateSlug function)
- Calculate next displayOrder via max()
- Insert, return created

**updateBeautyCategory(categoryId: string, data: { name?: string })**
- Auth gate, UUID validation
- If name provided, regenerate slug
- Update, return updated

**deleteBeautyCategory(categoryId: string)**
- Auth gate, UUID validation
- Check if beautyProducts reference this category
- If products exist: throw "Cannot delete category with existing products. Move or delete the products first."
- Check isDefault: if 1, throw "Cannot delete default categories"
- Delete, return { success: true }
