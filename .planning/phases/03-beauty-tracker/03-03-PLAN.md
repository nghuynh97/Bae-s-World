2. Create `src/actions/routines.ts` with these Server Actions:

**getRoutinesWithSteps()**
- Auth gate: createClient() -> getUser() -> throw "Unauthorized"
- Fetch all routines ordered by displayOrder asc
- For each routine, fetch routineSteps joined with beautyProducts and images, ordered by stepOrder asc
- For each step, fetch the "thumb" imageVariant for the product's image
- Sign the variant paths using getSignedImageUrls (batch all paths across all routines into one call)
- Return structure:
```typescript
type RoutineWithSteps = {
  id: string;
  name: string;
  slug: string;
  steps: {
    id: string;
    stepOrder: number;
    product: {
      id: string;
      name: string;
      brand: string | null;
      thumbnailUrl: string; // signed URL for thumb variant
    };
  }[];
};
```

**addRoutineStep(routineId: string, productId: string)**
- Auth gate
- zod validate both UUIDs
- Calculate next stepOrder: select max(stepOrder) from routineSteps where routineId, add 1 (default 0 if no steps)
- Insert new routineStep
- Return created step

**removeRoutineStep(stepId: string)**
- Auth gate
- zod validate UUID
- Delete the routineStep row
- Do NOT renumber remaining steps (sort by stepOrder handles gaps)
- Return { success: true }

**reorderRoutineSteps(routineId: string, stepIds: string[])**
- Auth gate
- zod validate routineId as UUID, stepIds as array of UUID strings
- Loop through stepIds array, update each routineStep's stepOrder to its index:
```typescript
for (let i = 0; i < stepIds.length; i++) {
  await db.update(routineSteps)
    .set({ stepOrder: i })
    .where(and(eq(routineSteps.id, stepIds[i]), eq(routineSteps.routineId, routineId)));
}
```
- Return { success: true }

1. Create `src/components/beauty/routine-step.tsx`:

"use client" component using @dnd-kit/sortable.

```typescript
"use client";

import { useSortable } from "@dnd-kit/sortable";
import { CSS } from "@dnd-kit/utilities";
import { GripVertical, X } from "lucide-react";
import Image from "next/image";

interface RoutineStepProps {
  step: {
    id: string;
    stepOrder: number;
    product: {
      id: string;
      name: string;
      brand: string | null;
      thumbnailUrl: string;
    };
  };
  index: number;
  onRemove: (stepId: string) => void;
}

export function RoutineStep({ step, index, onRemove }: RoutineStepProps) {
  const { attributes, listeners, setNodeRef, transform, transition, isDragging } = useSortable({
    id: step.id,
  });

  const style = {
    transform: CSS.Transform.toString(transform),
    transition,
  };

  return (
    <div
      ref={setNodeRef}
      style={style}
      className={`flex items-center gap-3 py-3 px-2 rounded-md ${
        isDragging ? "shadow-md bg-surface opacity-90 z-10" : "hover:bg-hover"
      }`}
    >
      <button
        {...attributes}
        {...listeners}
        className="cursor-grab active:cursor-grabbing touch-none p-1"
        aria-label="Drag to reorder"
      >
        <GripVertical className="h-5 w-5 text-muted-foreground" />
      </button>
      <span className="text-sm font-bold text-text-secondary w-6 text-center shrink-0">
        {index + 1}
      </span>
      <div className="h-10 w-10 rounded-md overflow-hidden shrink-0 bg-muted">
        {step.product.thumbnailUrl && (
          <img
            src={step.product.thumbnailUrl}
            alt={step.product.name}
            className="h-full w-full object-cover"
          />
        )}
      </div>
      <span className="text-sm font-body text-primary flex-grow truncate">
        {step.product.name}
      </span>
      <button
        onClick={() => onRemove(step.id)}
        className="p-1 text-muted-foreground hover:text-destructive shrink-0"
        aria-label={`Remove ${step.product.name}`}
      >
        <X className="h-4 w-4" />
      </button>
    </div>
  );
}
```

For each routine, render a card:
```tsx
<div className="bg-surface rounded-lg shadow-sm p-6 mb-6">
  <div className="flex items-center justify-between mb-4">
    <h3 className="font-display text-xl font-bold text-primary">{routine.name}</h3>
    <span className="text-xs text-text-secondary bg-muted px-2 py-1 rounded-full">
      {routine.steps.length} step{routine.steps.length !== 1 ? "s" : ""}
    </span>
  </div>

  {routine.steps.length === 0 ? (
    <p className="text-sm text-text-secondary text-center py-4">
      No steps yet. Search for products to add.
    </p>
  ) : (
    <DndContext sensors={sensors} collisionDetection={closestCenter} onDragEnd={(e) => handleDragEnd(e, routine.id)}>
      <SortableContext items={routine.steps.map(s => s.id)} strategy={verticalListSortingStrategy}>
        {routine.steps.map((step, index) => (
          <RoutineStep key={step.id} step={step} index={index} onRemove={handleRemoveStep} />
        ))}
      </SortableContext>
    </DndContext>
  )}

  <div className="mt-4">
    <RoutineStepSearch routineId={routine.id} onStepAdded={handleStepAdded} />
  </div>
</div>
```

handleDragEnd: Use arrayMove to reorder local state optimistically. Then call reorderRoutineSteps(routineId, newStepIds). On error: revert to previous order, toast "Something went wrong. Please try again."

handleRemoveStep: Remove step from local state optimistically. Call removeRoutineStep(stepId). Toast "[Product name] removed from [Routine name]" with no undo (simplify -- undo is complex with server state). On error: revert, toast error.

handleStepAdded: Call router.refresh() to refetch server data (simplest approach since we need the new step's signed thumbnail URL).
