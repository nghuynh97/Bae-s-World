1. **Imports:** `db` from `../src/lib/db`, all table schemas from `../src/lib/db/schema`, `createAdminClient` from `../src/lib/supabase/admin`.

2. **Storage cleanup function** `cleanStorage()`:
   - Use `createAdminClient()` to get admin Supabase client
   - List all files in `public-images` bucket under `portfolio/` prefix, then remove them all via `storage.from('public-images').remove([...paths])`
   - List all files in `private-images` bucket under `beauty/` prefix, then remove them all via `storage.from('private-images').remove([...paths])`
   - Log count of removed files. If bucket is empty or list returns empty, skip gracefully.

3. **Delete function** `resetAllData()` -- delete tables in this exact FK-safe order:
   ```
   await db.delete(routineSteps);
   await db.delete(portfolioItems);
   await db.delete(beautyProducts);
   await db.delete(imageVariants);
   await db.delete(images);
   await db.delete(routines);
   await db.delete(categories);
   await db.delete(beautyCategories);
   await db.delete(scheduleJobs);
   await db.delete(aboutContent);
   ```
   Do NOT delete `inviteCodes` or `profiles`.

4. **Seed portfolio function** `seedPortfolio(uploadedBy: string)`:
   - Create 3 categories: `{ name: 'Modeling', slug: 'modeling', displayOrder: 0, isDefault: 1 }`, `{ name: 'Travel', slug: 'travel', displayOrder: 1, isDefault: 0 }`, `{ name: 'Beauty', slug: 'beauty', displayOrder: 2, isDefault: 0 }`
   - Insert categories and capture returned IDs
   - For 20+ portfolio items: fetch cat images from `https://cataas.com/cat` (returns random cat image each time), upload each to Supabase Storage `public-images` bucket under path `portfolio/{uuid}.webp` via admin client `storage.from('public-images').upload(path, buffer, { contentType: 'image/jpeg' })`
   - For each uploaded image, create an `images` row with `{ originalName: 'cat-{i}.jpg', bucket: 'public-images', folder: 'portfolio', width: 800, height: 600, format: 'jpeg', uploadedBy }`
   - Create one `imageVariants` row per image: `{ imageId, variantName: 'medium', width: 800, height: 600, storagePath: 'portfolio/{uuid}.webp', sizeBytes: buffer.length }`
   - Create `portfolioItems` rows distributing across the 3 categories (round-robin), with titles like `'Cute Cat {i}'` and empty description
   - Create one `aboutContent` row with default empty values

5. **Seed beauty function** `seedBeauty(uploadedBy: string)`:
   - Create 4 beauty categories: Skincare (slug: skincare), Makeup (slug: makeup), Haircare (slug: haircare), Body Care (slug: body-care) -- all `isDefault: 1`
   - Create 8-10 beauty products with Vietnamese-style names: e.g., "Kem Chong Nang UV", "Son Mac Ruby Woo", "Serum Vitamin C", "Dau Goi Tresemme", "Sua Rua Mat CeraVe", "Phan Phu Innisfree", "Kem Duong Am Laneige", "Tay Trang Bioderma"
   - For each product: upload a cat image from `https://cataas.com/cat` to `private-images` bucket under `beauty/{uuid}.webp`, create `images` + `imageVariants` rows
   - Create `beautyProducts` rows with `{ name, brand: 'Brand Name', categoryId, rating: random 3-5, notes: '', isFavorite: random 0 or 1, imageId }`
   - Create 2 routines: `{ name: 'Morning', slug: 'morning', displayOrder: 0 }` and `{ name: 'Evening', slug: 'evening', displayOrder: 1 }`
   - Create 3-4 `routineSteps` per routine referencing some of the created beauty products

6. **Seed schedule function** `seedSchedule()`:
   - Create 15-20 jobs spread across 3 months (current month, previous month, next month)
   - Vietnamese client names: "Chi Lan", "Chi Huong", "Anh Minh", "Chi Thao", "Chi Mai", "Anh Tuan", "Chi Linh", "Chi Nga"
   - Locations: "Quan 1", "Quan 3", "Quan 7", "Thu Duc", "Binh Thanh", "Phu Nhuan", "Tan Binh", "Go Vap"
   - Time slots: start between 08:00-16:00, end 2-4 hours after start
   - Pay amounts: between 500000 and 3000000 VND (integer, multiples of 100000)
   - Status: mix of 'paid' and 'pending' (older jobs more likely paid)
   - `jobDate` format: 'YYYY-MM-DD' string

7. **Main function:**
   - Get first profile's `authId` via `db.select().from(profiles).limit(1)` for the `uploadedBy` field
   - If no profiles found, log error and exit
   - Call `cleanStorage()`, `resetAllData()`, `seedPortfolio(authId)`, `seedBeauty(authId)`, `seedSchedule()`
   - Log summary: "Seeded: X portfolio items, Y beauty products, Z jobs"
   - `process.exit(0)` on success, `process.exit(1)` on error

8. **In package.json**, add script:
   ```
   "seed": "node --env-file=.env.local node_modules/tsx/dist/cli.mjs scripts/seed-all.ts"
   ```

**Important notes:**
- Use `crypto.randomUUID()` for generating storage paths (not for DB IDs -- let DB `defaultRandom()` handle those)
- Use `db.insert(table).values(data).returning()` to capture generated IDs for FK references
- Wrap each fetch to cataas.com in a try-catch; if fetch fails, use a 1x1 transparent placeholder buffer
- Add a small delay (100ms) between cataas.com fetches to avoid rate limiting
